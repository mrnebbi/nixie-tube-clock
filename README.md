# Nixie Tube Clock #


## Overview ##

The Nixie Tube Clock uses Arduino, Bluetooth and Nixie Tubes to create a retro looking bluetooth enabled alarm clock.
The final product will include and iOS app to set the time, date, and alarms.

---

## Pinout ##

### Adafruit NRF8001 ###

* REQ - Arduino D10
* RST - Arduino D9
* RDY - Arduino D2
* SCK - Arduino D52
* MISO - Arduino D50
* MOSI - Arduino D51

### Clock (DS3231) ###

* SDA - Arduino D20
* SCL - Arduino D21

### Power ###

* Positive - Arduino 5V
* Ground - Arduino GND
